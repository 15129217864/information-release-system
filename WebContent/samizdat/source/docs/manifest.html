General Manifest

This is a description of the files included in this distribution.

  • com: contains the Java source files, in four packages:
    • com.jconfig
    • com.jconfig.win
    • com.jconfig.mac
    • com.jconfig.nix

  • docs: contains the documentation, which is linked to from 'readme.html', and which is cross-linked. The root level of this folder contains general documentation
    • javadocs: documentation of the Java code, in PolarDoc format. These were built using the '-private' switch
    • cdocs: documentation of the native code, in the following folders:
      • common
      • win
      • mac
      • nix

  • files: contains the C++ source and header files, in the following folders
    • common
    • win
    • mac
    • nix

  • meta: Mac-only. Used by the CodeWarrior project to build the Mac native code. See macinstall.html for more information.

  • testbed: Windows-only. Contains an MSVC project used to test the Windows native code. See wininstall.html for more information.

  • *.bat: Windows-only batch files.
    • simpcomp.bat: builds the Java code
    • buildall.bat: builds all the Windows native code DLLs using nmake and the .mak files
    See wininstall.html for more information.

  • *.mak / *.mdp: Windows-only MSVC++ project files:
    • jcSNA.mak / jcSNA.mdp - builds jcnfigSN.dll (SunJDK with Win95/98)
    • jcSNW.mak / jcSNW.mdp - builds jcnfigSW.dll (SunJDK with WinNT)
    • R1A.mak / R1A.mdp - builds jcnfigMS.dll (MSVM1.5 with Win95/98)
    • R1W.mak / R1W.mdp - builds jcnfigMW.dll (MSVM1.5 with WinNT)
    • R2A.mak / R2A.mdp - builds jcnfigM2.dll (MSVM2+ with Win95/98)
    • R2W.mak / R2W.mdp - builds jcnfig2W.dll (MSVM2+ with WinNT)
    See wininstall.html for more information.

  • readme.html: Contains general information, and links to the other documentation.

  • mac_Project.sit: Mac-only. Contains a CodeWarrior project used to build the Mac native code in a Stuffit archive. See macinstall.html for more information.

  • mac_JConfig_rsrc.sit: Mac-only. Contains the JConfig resource file in a Stuffit archive. See macinstall.html for more information.

  • lxmake.sh: Linux-only. A shell script used to compile the Linux native code. See nixinstall.html for more information.

  • mm.sh: OSX-only. A shell script used to compile the OSX native code. See osxinstall.html for more information.

  • applestubs.zip: OSX-only. Stub files needed to compile some of the java classes on a non-OSX system. See osxinstall.html, nixinstall.html, wininstall.html, or macinstall.html for more information.
